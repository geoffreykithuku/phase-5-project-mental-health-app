## PROJECT DESCRIPTION
MindEase is a counselling web app that focuses on Suicide prevention therapy.It provides a platform where distressed patients can seek emotional support from counsellors on the platform.
## Technologies used
- HTML
- CSS
- JS
- React JS
- ruby on rails
